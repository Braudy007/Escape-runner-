# Fast Escape

Un jeu de course infini (Infinite Runner) développé en Java.

## Comment jouer
- **Flèches HAUT/BAS** : Naviguer dans le menu
- **ENTRÉE** : Valider / Commencer
- **ESPACE** : Sauter
- **P** : Pause
- **ECHAP** : Revenir au menu

## Comment lancer le jeu

### Prérequis
Vous devez avoir **Java (JDK 11 ou supérieur)** installé sur votre machine.

### Méthode simple (Windows)
Double-cliquez sur le fichier `run.bat`.

### Méthode manuelle (Ligne de commande)
1. Ouvrez un terminal dans ce dossier.
2. Compilez le code (si ce n'est pas fait) :
   ```bash
   javac -d bin -sourcepath src/main/java src/main/java/com/fastescape/*.java
   ```
3. Lancez le jeu :
   ```bash
   java -cp bin com.fastescape.Main
   ```
